iOSExamples

====

Sample code.

## Building

> __requires Xcode Version 9.0 (9A235)__

1. Install Xcode from [Apple](https://developer.apple.com/download/more/).

1. Open iOSExamples.xcodeproj in Xcode.

1. Build the iOSExamples scheme in Xcode.
