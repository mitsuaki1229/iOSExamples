iOSExamples

====

Sample code.

## Building

> **requires Xcode Version 12.1 (12A7403)**

1.  Install Xcode from [Apple](https://developer.apple.com/download/more/).
2.  Open iOSExamples.xcodeproj in Xcode.
3.  Build the iOSExamples scheme in Xcode.

## Target

-   iPhone
-   iPad
-   iOS ~> 12.0
